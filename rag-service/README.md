# OpenSpace RAG System\n\nThis directory contains the Retrieval-Augmented Generation (RAG) backend for the OpenSpace CRM.\n\n## Architecture\n- **Data Ingestion**: Document loaders, chunking, and embedding generation.\n- **Vector Store**: Vector database integrations (e.g. pgvector, Chroma).\n- **Retrieval**: Semantic search and re-ranking.\n- **Generation**: LLM prompt chains and streaming responses.\n- **API**: FastAPI endpoints.
